## [Task]

블로그 플랫폼의 게시글(Post) 기능을 Clean Architecture로 구현해줘.

---

## [비즈니스 요구 사항]

### 사용자가 할 수 있는 것

1. 게시글 작성 (처음에는 비공개 상태)
2. 게시글 수정 (비공개 상태일 때만)
3. 게시글 발행 (공개로 전환)
4. 게시글 삭제 (완전 삭제가 아닌 보관 처리)
5. 게시글 조회

### 제약 조건

- 로그인한 사용자만 글 작성 가능
- 하루에 최대 10개까지만 작성 가능
- 제목: 5~100자
- 본문: 10~10,000자
- 본인 글만 수정/삭제 가능 (관리자는 모든 글 가능)

---

## [기술 스택]

- TypeScript
- Next.js (npx create-next-app@latest)
- Shadcn UI
- MDX 콘텐츠 관리

---

## [출력]

- 기획한 클린 아키텍처 폴더 구조를 출력해줘.
